# Basic_SoC_Implementation
Soc 기본 블럭. Peripheral Macro 설계 및 RTL-to-GDSII

## 👍 SoC 102
SoC의 기본 구성을 이루는 기본적인 요소들을 학습하고 RTL-to-GDSII의 Flow를 따라 Logic을 Physical Implementation해 보아요!

## 👉 어서오세요
* SoC 102에서는 ASIC의 RTL-to-GDSII 흐름에 따라 SoC의 구성 요소를 칩으로 구현해 보아요. SoC의 기본지식을 하나하나 HDL을 통해 구현하고, 국회의원도가 있는 I/F를 물리적 구현으로 인해 어느 새 상징이 여러분의 삶 속으로 대신하게 되는 것을 느낄 수 있다는 것입니다.

## 👉 프로젝트 개요
* SoC의 기본을 이루는 RTL Block Coding
* 주요 Combinational / Sequential Logic의 HDL 설계 및 검증
* ASIC Flow에 사용하는 전문 EDA Tool을 사용한 Functional Verification
* RTL 소스의 Constraint 작성
* Contraint 적용한 RTL 소스의 Synthesis
* Synthesis 이후의 Gate Level Netlist에 대한 Post Synthesis
* Synthesis 이후의 Gate Leverl Netlist에 대한 Post Synthesis Simulation
* Gate Level Netlsit(GLN) 및  '.sdc', '.lef' 적용한 Auto Place and Route(APR)
